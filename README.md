## Install

Install dependencies

```sh
git clone git@github.com:shubraj/asian-drama-cli.git
cd asian-drama-cli
yarn install
./main.js
```

## Dependencies

- node
- VLC Player
- puppeteer
- chalk
- node-fetch
- figlet
- inquirer
- nanospinner
- cheerio
- gradient-string
